# Eclipse DataGrid Demos Repository

Welcome to the demos repository!

Here you will find different Eclipse DataGrid demo applications that showcase different ways of using Eclipse DataGrid.
All the demos can be deployed in the [DataGrid Cloud](https://cloud.microstream.one/)

[Storage Demo](./cluster-storage-demo) - This demo showcases eclipse-store with the Gigamap to provide a simple book
REST application.

[Hibernate Cache Demo](./cluster-hibernate-cache-demo) - This demo builds on the Storage Demo, replacing the storage
layer with a cache. Data is stored on a separate database server and cache invalidations are replicated among the cache
nodes.
